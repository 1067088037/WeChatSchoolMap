# 2021年高校微信小程序应用开发赛作品校园缩影

## 软件简介

本小程序“校园缩影”基于一个“会呼吸”的地图的理念将事件投影到地图之上，令地图是人现实中活动的变化而变化，将大学生在校园中行走、生活而学习到的生活经验投影到地图上，将陌生的地图变为熟人地图。本小程序特色功能分为两大块，活动与攻略，由新生对大学一些重要而无法在地图上显示出的攻略需求出发，融入了实时展现校园活动、失物招领等针对老生的需求，在地图上重点呈现。老生可以足不出户就洞察校园内又有什么新活动发生，并且查看活动的具体信息与推文，甚至可以关注未来的特定活动，随时随地云逛校园，新生可以通过攻略导航在陌生的校园里完成入学的准备工作。是一款专为大学生准备的地图。
